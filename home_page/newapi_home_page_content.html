Codex 安装指南

此流程 100% 使用官方原版安装包，确保服务体验与官方完全一致。

官方原版安装

100% 使用官方原版安装包。

Windows 系统要求

Windows 10（版本 1809 / build 17763）及以上。

macOS / Linux 系统要求

macOS 10.15+ 或任意现代 Linux 发行版。

概览 安装 Codex CLI 创建 .codex 目录 获取 API 密钥 创建 auth.json 创建 config.toml 重启并验证 开始使用
01

安装 Codex CLI

运行以下任意一条命令，全局安装官方 Codex CLI：

npm install -g @openai/codex # 或者 / or brew install codex

两条命令安装的是同一个包，选择适合你环境的即可。

02

创建 .codex 目录

macOS / Linux

在终端中执行以下命令：

rm -rf ~/.codex mkdir ~/.codex

Windows

若目录已存在请先删除，再重新创建（在文件资源管理器或 PowerShell 中操作）：

# PowerShell Remove-Item -Recurse -Force ~\.codex -ErrorAction SilentlyContinue New-Item -ItemType Directory ~\.codex

将 <your-username> 替换为你的实际 Windows 用户名。

03

获取 API 密钥

前往控制台的 「令牌管理」 页面，创建一个新密钥并复制。

04

创建 auth.json

macOS / Linux

在 ~/.codex 路径下，删除已有的 auth.json（若有），新建一个：

Windows

在 C:\Users\<your-username>\.codex 路径下，删除已有的 auth.json（若有），新建一个：

{ "OPENAI_API_KEY": "YOUR_API_KEY" }

将 YOUR_API_KEY 替换为上一步复制的密钥。

05

创建 config.toml

macOS / Linux

在 ~/.codex 路径下，删除已有的 config.toml（若有），新建一个：

Windows

在 C:\Users\<your-username>\.codex 路径下，删除已有的 config.toml（若有），新建一个：

model_provider = "OpenAI" model = "gpt-5.5" model_reasoning_effort = "xhigh" disable_response_storage = true preferred_auth_method = "apikey" [model_providers.OpenAI] name = "OpenAI" base_url = "https://new-api-latest-xhh3.onrender.com/v1" wire_api = "responses"

以下内容请原封不动粘贴，不要修改任何值。

06

重启终端 验证安装

重启终端后运行以下命令，确认安装成功：

codex -V

若输出版本号，则说明 Codex 已正确安装。

07

开始 使用 Codex

进入任意项目目录，启动 Codex：

# 切换到项目目录 / Navigate to project cd your-project-folder # 启动 Codex / Launch Codex codex

完全支持 VSCode 官方插件。
